## Featured Projects

### SkiliAir — Air Quality Monitoring System

Real-time air quality monitoring system using Raspberry Pi, sensors, MySQL, Flask, LCD output, alerts, and a live web dashboard.

**Technologies:** Python, Flask, MySQL, Raspberry Pi, GPIO, Chart.js

---

### MATT — Web-Controlled Writing Robot

Web-controlled robot system using Firebase, ESP32, UART communication, and an embedded KL25Z controller.

**Technologies:** ESP32, Firebase, UART, C/C++, JavaScript

---

### KL25Z Mini Radar

Radar-style distance scanner using ultrasonic sensing, servo movement, UART data streaming, and Python visualization.

**Technologies:** FRDM-KL25Z, C, UART, Python, Matplotlib

---

### C++ Object-Oriented Programming Projects

Academic projects involving classes, inheritance, polymorphism, file handling, matrices, and fractions.

**Technologies:** C++, OOP, UML, File I/O
